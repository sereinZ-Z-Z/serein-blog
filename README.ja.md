# Serein Blog

This is my personal blog site, customized from an existing Astro blog template.

## Notes

- This site is for personal learning notes, life records, and blog posts.
- Upstream template promotional and collaboration content has been removed to avoid misattribution.
- Required third-party license files are kept according to open-source license requirements.

## Development

```bash
pnpm install
pnpm dev
pnpm build
```
